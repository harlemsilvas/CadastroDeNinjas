# Code of Conduct

## Nosso Compromisso
Como contribuidores e mantenedores deste projeto, nós nos comprometemos a criar um ambiente acolhedor, inclusivo e respeitoso para todos.

## Padrões Esperados
- Seja respeitoso e profissional.
- Evite linguagem ou comportamento abusivo, discriminatório ou ofensivo.
- Receba críticas de forma construtiva e ofereça feedback de forma respeitosa.

## Procedimento para Reportar
Se você experienciar ou testemunhar comportamento que viole este Código de Conduta, por favor entre em contato com os mantenedores do projeto através do e-mail: harlem.claumann@gmail.com

## Aplicação
Os mantenedores do projeto se reservam o direito de investigar e agir sobre qualquer queixa, inclusive removendo, editando ou rejeitando contribuições ou banindo usuários quando apropriado.

## Agradecimento
Agradecemos por fazer deste projeto um espaço seguro e produtivo para todos.
